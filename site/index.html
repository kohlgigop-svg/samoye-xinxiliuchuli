GitHub Pages + Supabase

表格任务匹配分析台

这个站点可直接部署到 GitHub Pages。前端负责上传、读取和分析；后端建议用 Supabase 充当数据仓库，仅存储原始表格文件和轻量元数据，真正读取时再按需解析。

支持 xlsx / xls / csv 支持多 json / jsonl 未配置后端时可本地演示
